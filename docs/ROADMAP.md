# Roadmap

## Vision
To become the ultimate "Learning Operating System" that bridges the gap between passive consumption of information and active mastery of knowledge.

## Mission
Provide a low-friction, high-intent knowledge capture tool that leverages spaced repetition to guarantee long-term retention.

## Design Principles
- **Intentionality**: UI surfaces only what is needed for the current action.
- **Minimality**: Remove navigation friction. Capture is the priority.
- **Performance**: High reactivity with optimistic UI updates.

## Version 1: Foundation (Completed)
- [x] Project/Category/Topic Hierarchy
- [x] Learning Log with inline creation
- [x] Spaced Repetition Scheduling
- [x] Dashboard and Timeline views

## Version 2: Active Recall
- [ ] Flashcards
- [ ] Question Bank
- [ ] Topic Knowledge Hub
- [ ] Global Search

## Version 3: Intelligence
- [ ] AI Tutor integration (LLM-based summarization)
- [ ] Knowledge Graph visualization
- [ ] Smart Revision (Adaptive intervals)
- [ ] Weak Topic Detection

## Version 4: Scaling
- [ ] User Collaboration / Shared Projects
- [ ] Deep Analytics & Insights
- [ ] Multi-device Sync

## Long-term Vision
The platform evolves from a personal log into an autonomous personal Knowledge Operating System, where insights are surfaced, connections are mapped, and the user's focus is automatically directed to their weakest knowledge areas.
